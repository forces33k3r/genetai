Getting Started with GenetAi

Welcome to GenetAi documentation. This guide will help you integrate with our genetic analysis platform.

Quick Start

To get started with GenetAi API:

// Install GenetAi SDK
npm install genetai-sdk

// Initialize the client
const { GenetAi } = require('genetai-sdk');
const client = new GenetAi({
    apiKey: 'your-api-key',
    environment: 'production'
});

// Upload DNA data
const result = await client.uploadDnaData({
    file: 'path/to/dna_data.txt',
    analysisType: 'full'
});
